Unleashing Humanoid Reaching Potential via Real-world-Ready Skill Space

저자: Zhikai Zhang, Chao Chen, Han Xue, Jilong Wang, Sikai Liang, Yun Liu, Zongzhang Zhang, He Wang, Li Yi | 날짜: 2025-05-16 | URL: https://arxiv.org/abs/2505.10918

Essence

본 논문은 휴머노이드 로봇의 대규모 도달 공간을 활용한 전신 제어(WBC) 문제를 해결하기 위해 Real-world-Ready Skill Space (R2S2)를 제안한다. 실제 환경 적용 가능한 기본 스킬들을 라이브러리로 구성하고 이를 통합 잠재 공간으로 앙상블하여 효율적이고 sim2real 전이 가능한 구조적 스킬 사전을 제공한다.

Motivation

  • Known: 최근 강화학습 기반 휴머노이드 전신 제어 연구(AMO, HOMIE)가 진행되었으나, 이들은 복잡한 보상 공학 또는 궤적 최적화에 의존하여 최적화 어려움과 시뮬레이션-실제 환경 간 전이 문제가 발생한다.
  • Gap: 기존 접근법들은 다양한 스킬을 동시에 학습할 때 스킬 간 조정 및 전이가 불충분하며, 서로 다른 커맨드 공간의 불일치로 인해 통합 표현이 부족하다.
  • Why: 휴머노이드가 인간 수준의 대규모 도달 공간을 활용하여 높이와 거리가 다양한 객체와 상호작용할 수 있어야 일상 작업 보조가 가능하며, 이는 기저 위치 조정, 신체 자세 변화, 끝단 포즈 제어 등 복합적 스킬을 요구한다.
  • Approach: 실제 환경 적용 가능한 기본 스킬 라이브러리를 먼저 구성한 후, heterogeneous skill ensembling을 통해 CVAE 기반 학생 정책이 사전 학습된 교사 정책들의 지식을 상속받으면서 새로운 스킬 조정을 탐색하도록 한다. 이후 R2S2에서 샘플링하는 고수준 플래너를 학습하여 실제 목표 도달 과제를 수행한다.

Achievement

  • 구조적 스킬 사전 설계: 실제 환경 적용 가능한 기본 스킬들로부터 구성된 R2S2가 최소한의 보상 공학으로 복잡한 전신 제어 과제 실행을 지원
  • Heterogeneous skill ensembling 제안: IL과 RL을 동적으로 결합하여 독립적으로 훈련된 스킬들의 지식을 상속받으면서 스킬 간 조정 및 전이 능력을 확장하는 CVAE 기반 통합 신경 스킬 표현 개발
  • 실제 배포 달성: Unitree G1(29 DOF)과 Unitree H1(1.8m) 플랫폼에서 다양한 자율 목표 도달 과제에 대해 zero-shot sim2real 전이 성공
  • 전신 텔레오퍼레이션 시스템: 부수적 효과로 대규모 도달 공간을 활용한 휴머노이드 전신 텔레오퍼레이션 시스템 구현

How

  • 기본 스킬 라이브러리 구성: 로코모션, 신체 자세 조정 등 과제 불가지론적(task-agnostic) 기본 스킬들을 개별적으로 훈련하고 sim2real 평가를 통해 실제 환경 적용 가능성 보장
  • Heterogeneous skill training 환경 구성: 서로 다른 커맨드 공간을 가진 기본 스킬들을 통합하기 위한 환경 설계
  • CVAE 기반 학생 정책 훈련: Imitation Learning과 Reinforcement Learning을 동적으로 결합하여 고유 상태에 대한 모터 스킬 분포를 신경망으로 모델링
  • 고수준 플래너 학습: R2S2에서 스킬을 샘플링하도록 훈련된 플래너가 자율적으로 복잡한 전신 제어 과제 수행
  • 다중 플랫폼 검증: 서로 다른 DOF와 크기의 휴머노이드에서 일반화 가능성 평가

Originality

  • 실제 환경 적용 가능한 기본 스킬들로부터 스킬 공간 구성하는 새로운 접근으로, 기존 인간 모션 캡처 기반 스킬 공간과 차별화
  • Heterogeneous skill ensembling의 개념 도입으로, 독립적으로 훈련된 이질적 스킬들을 통합하면서 각 스킬의 실제 환경 적용 가능성 보장
  • CVAE를 사용한 신경 스킬 표현이 기존 계층적 제어 프레임워크의 주 커맨드 공간 기반 계획보다 효율적임을 입증
  • IL과 RL을 동적으로 결합하는 훈련 전략으로 기존 고정된 학습 방식 개선

Limitation & Further Study

  • 현재 방법은 주로 대규모 도달 공간이 필요한 과제에 초점을 두어 다른 유형의 복잡한 전신 제어 과제로의 확장 가능성 미검증
  • Heterogeneous skill training 환경 구성이 여전히 특정 과제에 맞게 설계되어야 하므로, 완전히 자동화된 환경 구성 방법 부재
  • 기본 스킬 라이브러리의 스킬 수와 다양성이 최종 성능에 미치는 영향에 대한 상세 분석 부족
  • 실제 환경에서의 배포 시 환경 변화(예: 다양한 지표면, 동적 장애물)에 대한 강건성 평가 제한적
  • 후속 연구로는 다양한 과제 유형으로의 확장, 자동화된 스킬 라이브러리 구성 방법 개발, 더 복잡한 조작 과제로의 적용 필요

Evaluation

Novelty: 4/5 Technical Soundness: 3/5 Significance: 4/5 Clarity: 4/5 Overall: 4/5

총평: 본 논문은 실제 환경 적용 가능한 기본 스킬들로부터 통합 스킬 공간을 구성하는 새로운 접근으로, 복잡한 휴머노이드 전신 제어 문제에서 보상 공학의 부담을 줄이면서 효과적인 sim2real 전이를 달성했다는 점에서 상당한 기여가 있다. 다만 방법의 일반화 범위와 자동화 수준에 대한 추가 검증이 필요하다.
