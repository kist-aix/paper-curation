DemoHLM: From One Demonstration to Generalizable Humanoid Loco-Manipulation

저자: Yuhui Fu, Feiyang Xie, Chaoyi Xu, Jing Xiong, Haoqi Yuan, Zongqing Lu | 날짜: 2025-10-13 | DOI: 10.48550/arXiv.2510.11258

Essence

DemoHLM은 단일 시뮬레이션 시연으로부터 수백~수천 개의 합성 궤적을 생성하여 휴머노이드 로봇의 일반화 가능한 로코-조작 정책을 학습하는 프레임워크이다. 계층적 제어 구조와 object-centric 동작 재현을 결합하여 시뮬레이션에서 실제 로봇으로의 제로샷 전이를 실현한다.

Motivation

  • Known: 휴머노이드 로봇의 전신 제어 기술은 상당히 진전되었으나, 로코-조작은 여전히 과제 특화 설계나 비용이 많이 드는 실제 데이터 수집에 의존하고 있다. MimicGen 같은 고정 베이스 조작기용 데이터 생성 방법이 존재한다.
  • Gap: 기존 RL 기반 휴머노이드 로코-조작 접근법은 과제별 리워드 함수 설계에 의존하거나 SMPL 휴머노이드 사용으로 실제 로봇 전이에 실패하며, 실제 데이터 기반 방법은 대규모 텔레옵 데이터 수집의 병목을 해결하지 못한다.
  • Why: 로코-조작의 일반화 가능한 정책 학습은 휴머노이드 로봇이 인간 환경에서 다양한 상호작용을 수행하기 위한 핵심이며, 시뮬레이션 기반 데이터 생성 파이프라인의 개발은 실제 로봇 배포의 확장성과 효율성을 크게 향상시킨다.
  • Approach: DemoHLM은 RL로 훈련된 저수준 whole-body controller와 imitation learning으로 학습된 고수준 조작 정책을 계층적으로 결합한다. 단일 시연을 3단계(로코모션, 사전-조작, 조작)로 분할하고, 임의의 초기 로봇 및 객체 포즈에서 high-level 명령을 변형하여 행동 시퀀스를 재현함으로써 다양한 합성 궤적을 생성한다.

Achievement

  • 시뮬레이션 기반 데이터 생성 파이프라인: 단일 시연으로부터 수백~수천 개의 성공적 궤적을 자동으로 합성하여 모방 학습 정책의 효과적 일반화를 가능케 함
  • 계층적 제어 구조: whole-body controller와 object-centric motion planning을 통합하여 고품질의 휴머노이드 로코-조작 데이터 생성
  • 강력한 시뮬-투-리얼 전이: Unitree G1 실제 로봇에서 10가지 로코-조작 과제 전반에 걸쳐 시뮬레이션과 비교 가능한 성능 달성
  • 데이터 효율성 검증: 합성 데이터 량의 증가가 정책 성능과 공간적 변화에 대한 일반화를 지속적으로 향상시킴을 실증

How

  • VR 기기로부터 캡처된 인간 포즈를 whole-body controller에 전송하여 시뮬레이션에서 단일 시연 수집
  • 시연을 로코모션, 사전-조작, 조작 3단계로 분할
  • 다양한 초기 로봇 포즈와 객체 위치에 대해 각 단계의 high-level 명령을 object-centric 방식으로 수정
  • 수정된 명령 시퀀스를 whole-body controller로 재현하여 새로운 궤적 생성
  • 모든 성공한 합성 궤적으로 behavior cloning 정책 훈련
  • 학습된 정책을 실제 로봇에 RGB-D 카메라 기반 시각 피드백과 함께 배포

Originality

  • 고정 베이스 조작기용 MimicGen의 개념을 휴머노이드 로코-조작으로 확장하여, 전신 밸런스 유지 하에서 end-effector 궤적 재현이라는 새로운 기술적 과제 해결
  • whole-body controller와 object-centric action replay의 결합을 통해 계층적 제어와 데이터 생성의 시너지 달성
  • 시뮬레이션 기반 다양한 궤적 생성으로 실제 로봇 데이터 수집 병목 해결하면서도 강력한 sim-to-real 전이 실현
  • 단일 시연으로부터 시작하는 데이터 효율적 접근으로 기존 RL 기반 과제 특화 설계의 한계 극복

Limitation & Further Study

  • 전신 컨트롤러의 성능 상한이 전체 파이프라인의 상한을 제한할 수 있으며, 복잡한 동역학 상호작용을 요구하는 과제에서의 확장성 미검증
  • object-centric action replay의 정확도는 초기 시연의 품질에 크게 의존하며, 근본적으로 다른 새로운 과제 카테고리로의 확장 방법이 명확하지 않음
  • 현재 RGB-D 카메라 기반 시각 피드백만 실험되었으며, 다른 센서 모달리티나 시각적 변화(조명, 가림 등)에 대한 강건성 평가 부족
  • 10개 과제의 평가가 주로 객체 조작 중심이며, 더 복잡한 동작 조합이나 장기간 계획을 요구하는 과제에 대한 성능 미확인
  • 후속연구: (1) 전신 컨트롤러 성능 향상 또는 적응형 제어 메커니즘 개발, (2) 시연 품질 자동 평가 및 개선 방법, (3) 다중 모달 센서 통합 및 환경 변화 대응 강건성 향상

Evaluation

Novelty: 4/5 Technical Soundness: 3/5 Significance: 4/5 Clarity: 4/5 Overall: 4/5

총평: DemoHLM은 휴머노이드 로코-조작의 일반화 가능한 정책 학습을 위한 실용적이고 확장 가능한 프레임워크를 제시하며, 단일 시연 기반 합성 데이터 생성과 강력한 실제 로봇 전이를 통해 시뮬레이션-투-리얼 격차를 효과적으로 해결한다. 계층적 제어 구조의 활용과 10개 과제 검증으로 실용성을 입증하였으나, 복잡한 동역학 상호작용과 환경 변화에 대한 추가 강건성 평가가 향후 개선 과제이다.
